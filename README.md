## üìñ Project Overview

**FixOnGo** is a cross-platform mobile application developed for drivers facing on-road vehicle breakdowns. It combines **AI-powered troubleshooting**, **GPS location services**, **mechanic & tool requests**, **real-time tracking**, **in-app chat**, **call center support**, and **secure digital payments** to deliver fast, reliable, and stress-free roadside assistance.

Developed as a **Mobile Application Development** group project (Group 01) at **NSBM Green University Town**.

## ‚ú® Key Features

- **User-Friendly Interface** ‚Äî Intuitive design with light & dark themes
- **AI-Based Breakdown Assistance** ‚Äî 24/7 rule-based chatbot for common issues (battery, tire, fuel, overheating, etc.)
- **Location Tracking & Mapping** ‚Äî Real-time GPS + Google Maps showing nearby mechanics
- **Mechanic & Tool Request System** ‚Äî Request verified professionals or spare parts/tools with ETA & cost estimation
- **Call Center Support** ‚Äî Direct in-app connection to expert guidance
- **In-App Communication** ‚Äî Real-time chat between users and service providers
- **Payment Gateway Integration** ‚Äî Secure payments (PayHere / Stripe) with transparent pricing
- **Service Provider Database** ‚Äî Verified mechanics, garages, towing services with ratings & reviews

## üö´ Out of Scope (as per proposal)

- Physical repair operations
- Advanced hardware-based diagnostics
- Insurance claim handling
- International assistance
